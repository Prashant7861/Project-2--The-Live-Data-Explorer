Project Description

CINEMATIC – Movie Explorer & Finnish Cinema Finder is a modern, responsive web application designed to explore movies and real-time cinema schedules using asynchronous JavaScript and external APIs.

The application integrates two different data sources to provide a complete user experience:

OMDb API (JSON)for global movie search, including titles, posters, ratings, and detailed metadata
Finnkino API (XML – simulated with structured mock data)** for real-time Finnish cinema showtimes, including theatres, screening times, and movie details

The main objective of this project is to demonstrate practical understanding of AJAX (Asynchronous JavaScript and XML) concepts, including:

* Fetching data using the (Fetch API with async/await)
* Handling **multiple data formats (JSON and XML-like structures)
* Dynamically updating the **DOM without page reloads

The application features a dual-mode interface where users can switch between:

Search Mode find movies globally using OMDb
Cinema Mode browse Finnish cinema schedules using Finnkino data

To overcome real-world limitations (such as API restrictions), the Finnkino integration uses a mock dataset that mirrors the actual XML structure, ensuring that the application logic remains realistic and production-relevant.

Additionally, the project emphasizes:

Responsive design using CSS Grid and Flexbox
Interactive UI/UX, including modals, animations, and loading states
Performance optimization through caching and efficient rendering
Accessibility features, such as keyboard navigation and focus management

Overall, this project showcases the ability to build a dynamic, data-driven web application that combines multiple APIs, handles different data formats, and delivers a smooth and engaging user experience.
